# Quick Start Guide

## Installation

1. Make sure you have Node.js installed (version 16 or higher)
2. Open a terminal in this directory
3. Run: `npm install`
4. Run: `npm run dev`
5. Open your browser to the URL shown in the terminal (usually http://localhost:5173)

## First Use

1. **User Mode** - For communication
   - Click "User Mode" from the home page
   - Choose Icons or Keyboard interface
   - Click icons or type to build your message
   - Click "Send" to save to message history
   - Click "Speak" to hear your message

2. **Admin Mode** - To customize
   - Click "Admin Mode" from the home page
   - Add/edit/delete icons and categories
   - All changes are saved automatically

## Features Overview

### Icons Interface
- Large, colorful icon buttons organized by category
- Filter by category using the category buttons at the top
- Click icons to build your message word by word

### Keyboard Interface
- Full QWERTY keyboard layout
- Quick word buttons for common words
- Space, backspace, and send buttons

### Admin Features
- **Icons Tab**: Manage all icons (add, edit, delete)
- **Categories Tab**: Manage categories with custom colors

## Tips

- Messages are saved in your browser's localStorage
- You can click on message history items to reuse them
- The app works offline after first load (PWA-ready)
- Works on tablets, phones, and desktop computers
